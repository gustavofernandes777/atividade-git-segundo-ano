# Atividade de revisão
Um pequeno mercado deseja criar um sistema simples para controlar os produtos em estoque.
## O programa deverá:
- Cadastrar produto, quantidade e preço;
- Informar a situação do estoque;
- Exibir um relatório final com todos os produtos.
## Requisitos do Programa
### Parte 1 – Cadastro
- Solicite o nome, preço e a quantidade do produto;
- Utilize uma lista para armazenar os dados do produto;
- Utilize uma matriz para armazenar todos os produtos
## Parte 2 – Processamento
- Para cada produto, solicite:
- Nome;
- Preço;
- Quantidade em estoque;
- Verifique a situação do produto no estoque:
- Estoque Alto → quantidade maior que 50;
- Estoque Médio → quantidade entre 20 e 50;
- Estoque Baixo → quantidade menor que 20.
# Parte 3 – Relatório Final
- Exiba:
- Nome do produto;
- Quantidade;
- Preço;
- Situação do estoque.
