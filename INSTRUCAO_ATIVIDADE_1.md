# ATIVIDADE 1
Um professor deseja criar um programa que:
* Peça ao aluno para informar quantas notas deseja cadastrar.
* Leia todas as notas (valores de 0 a 10).
* Caso o usuário digite uma nota inválida (menor que 0 ou maior que 10), o programa deve pedir novamente até que seja válida.

Ao final, o programa deve:
* Calcular a média.
* Informar se o aluno está:
* Aprovado (média ≥ 7)
* Recuperação (média entre 5 e 6.9)
* Reprovado (média < 5)
